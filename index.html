HTML y CSS

Estás viendo una página de texto en gris oscuro sobre gris claro. El primer texto, que destaca por su tamaño, peso y color, es un encabezado de primer nivel, al que se le asignó un color rojo.

El estilo fue asignado con CSS. Mientras que contenido y jerarquía se deben al HTML. Pero con estos dos lenguajes de descripción no podemos incluir interactividad dinámica en la página. Para poder hacerlo, es necesario utilizar un lenguaje de programación.

JavaScript

1. Desplegar una alerta con un click

Tomé el elemento <button>…</button>, le agregué un atributo "OnClick", que en su variable contiene programación en JavaScript con la que se desplegará, en el click, una alerta con un mensaje que resulta de concatenar una variable con un texto entre comillas.

2. Jugar con la consola de javascript

Utiliza el "shortcut" de teclado que corresponda:

  • Si estás usando Chrome en PC, presiona Ctrl + Shift + J. Si estás usando el mismo navegador en Mac, presiona Cmd + Alt + J
  • Si estás usando Firefox en PC, presiona Ctrl + Shift + K. Si estás usando el mismo navegador en Mac, presiona: Cmd + Alt + K
  • Si estás usando Safari, presiona: Cmd + Alt + C

Cuando aparezca la consola podrás leer un mensaje que resulta de concatenar una variable ya utilizada con otro texto entre comillas.

En la consola también puedes escribir algo. Intenta copiando, pegando, modificando y presionando Enter:

var sujeto = "Lorem";
var adjetivo = "ipsum";
console.log(sujeto +" es muy "+ adjetivo);

Lo que hiciste fue crear variables, a las que asignaste "cadenas de caracteres". Pero no tendrías que ocupar comillas de utilizar números:

var primervalor = 20;
var segundovalor = 30;
console.log(primervalor + segundovalor);

Antes de que presiones Enter, una pregunta: ¿Qué crees que resultará, 50 o 2030?

En caso que no le hayas acertado, podrías tener más suerte con lo que sigue: Piensa un número entre 1 y 10.

console.log(Math.floor((Math.random() * 10) + 1));

Tus opciones partían en 1, incrementándose tantas veces como lo permitiese la condición de menor o igual a 10:

for (var x=1; x<=10; x++){ console.log(x); }

3. Manipular el DOM

¿Te acuerdas de cuánto costaba el programa de gobierno de Ossandón?

Si revisas el código fuente de la página, notarás que el párrafo de identidad "cifra" está vacío. El texto está como puesto allí con JavaScript. Lo mismo pasa más arriba, con el encabezado de primer nivel.

Manipular el DOM es como "photoshopear" una imagen. Si capturaste 3 elementos, y con Photoshop le agregas un cuarto, en ningún caso estás modificando lo que fue, en principio, capturado.
