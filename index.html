Algunas pruebas

HTML y CSS

Estás viendo una página con un color de fondo gris claro, y texto en gris oscuro.

El primero texto destaca, por su tamaño, peso y color. Se trata de un encabezado de primer nivel, al que se le asignó un color rojo.

Lo que ves, como estilo, se debe a las reglas CSS. Lo que ves, como contenido y jerarquía, se debe a los elementos HTML.
